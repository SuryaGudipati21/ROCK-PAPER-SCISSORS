# Implementation Guide — Rock-Paper-Scissors Game

This guide explains how to set up and run the **Rock-Paper-Scissors Game**, the basic version.

---

## Step 1 — Download or Clone the Project
You can either:
- Download the ZIP file from GitHub

---

## Step 2 — Run the Game
To play the game:

- Locate **index.html**
- Double-click to open it
- It will launch in any modern browser (Chrome, Firefox, Edge, etc.)

This loads the **Rock-Paper-Scissors game page**.

---

## Common Problems & Solutions

### Scores not showing in results page
- Play at least one round  

---

### Images not loading
- Ensure the **pictures/** folder is next to `Rock-Paper-Scissors.html`  
- Confirm file names exactly (case-sensitive):  
    - `rock-emoji.png`  
    - `paper-emoji.png`  
    - `scissors-emoji.png`  
- Verify images are committed if using GitHub Pages
